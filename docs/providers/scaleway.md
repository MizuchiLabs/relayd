# Scaleway

```env
RELAYD_PROVIDER_SCALEWAY_TYPE=scaleway
RELAYD_PROVIDER_SCALEWAY_TOKEN=your-scaleway-secret-key
RELAYD_PROVIDER_SCALEWAY_ORGANIZATION_ID=your-organization-id # Optional, depending on your account setup
RELAYD_PROVIDER_SCALEWAY_ZONES=example.com
```

- **Scope**: `public`
- **Requires**: A Scaleway Secret Key (and potentially Organization ID).
