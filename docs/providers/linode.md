# Linode

```env
RELAYD_PROVIDER_LINODE_TYPE=linode
RELAYD_PROVIDER_LINODE_TOKEN=your-linode-personal-access-token
RELAYD_PROVIDER_LINODE_ZONES=example.com
```

- **Scope**: `public`
- **Requires**: A Linode Personal Access Token (can be generated in the [Linode Cloud Manager](https://cloud.linode.com/profile/tokens)).
