# Namecheap

```env
RELAYD_PROVIDER_NAMECHEAP_TYPE=namecheap
RELAYD_PROVIDER_NAMECHEAP_USER=your-namecheap-username
RELAYD_PROVIDER_NAMECHEAP_TOKEN=your-namecheap-api-key
RELAYD_PROVIDER_NAMECHEAP_ZONES=example.com
```

- **Scope**: `public`
- **Requires**: Namecheap Username and API Key.

**Note:** You must first whitelist your public IP address in the Namecheap console before you can successfully use their API.
