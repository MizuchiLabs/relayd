# RFC2136 (BIND, Technitium)

```env
RELAYD_PROVIDER_BIND_TYPE=rfc2136
RELAYD_PROVIDER_BIND_SCOPE=local
RELAYD_PROVIDER_BIND_SERVER=192.168.1.5:53
RELAYD_PROVIDER_BIND_KEY_NAME=tsig-key
RELAYD_PROVIDER_BIND_KEY=base64-encoded-key
RELAYD_PROVIDER_BIND_ZONES=home.local
```

- **Scope**: `local`
- **Requires**: TSIG key configuration on your DNS server.
